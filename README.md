## 국민대학교 커뮤니티

학교 내 커뮤니티 서비스를 실시하여 국민대학교 내 학생 및 직원들이 서로 쉽게 소통하고 정보를 얻을 수 있게끔 함. 웹 사이트 형식이 기본이고 모바일에서도 간편하게 사용할 수 있게끔 안드로이드 어플리케이션도 같이 개발함.
